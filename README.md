# `twlist2list`

Bash script to move users on a Twitter list to your list

## Usage

```shellsession
$ ./twlist2list <list id to retrieve> <your list id to append>
```

## Requirement

[stedolan/jq](https://github.com/stedolan/jq), [twitter/twurl](https://github.com/twitter/twurl)
